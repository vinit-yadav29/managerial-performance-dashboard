# 📊 Managerial Performance Dashboard

## 🚀 Overview
This project analyzes managerial performance using Python, Pandas, and Streamlit.

## 📌 Features
- Data Cleaning & Preprocessing
- KPI Metrics (Performance, Satisfaction, Revenue)
- Interactive Dashboard with Filters
- Correlation Analysis

## 🛠 Tech Stack
- Python
- Pandas
- Seaborn
- Matplotlib
- Streamlit

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
